基础SVG图形

绘制SVG流程

  1. 打开SVG标签
  2. 绘制基本图形/文字,设置基本属性
  3. 添加滤镜.动画等等附加属性
矩形 : rect
圆形 : circle
椭圆 : ellipse
直线 : line
多边形:希腊语[poly==many] [gon==angle]
曲线:[poly line]
路径 : path
SVG与文本结合 :
text (1.可以使用tspan进行文字内联操作 2.使用defs定义路径,textpath绘制文本路径 3.可以绘制链接文本)

SVG基本属性

属性:Stroke
属性:滤镜filter 多种滤镜种类
颜色渐变(gradient)-线性:linear
颜色渐变(gradient)-放射性:radial
